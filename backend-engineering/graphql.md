# GraphQL

## What is GraphQL?
- GraphQL is a query language for an API.
- It provides an independent layer between the server and the client.
- The client communicates with a single GraphQL endpoint and changes the query to select with data it wants.
- The server resolves the GraphQL queries to give the client the data it needs.

## Why use GraphQL?
- While REST APIs can have data overfetching and underfetching problems, GraphQL doesn't have that.
- GraphQL can help rapid client-side updates since the backend doesn't have to update accordingly.
- By monitoring requests, analytics can show what kind of data is mostly accessed by clients.
