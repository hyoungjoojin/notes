# Apache Kafka

- Apache Kafka is a distributed event-streaming platform that can handle massive pipelines of real-time data.

## Producer API
- The producer of Kafka produces streams of data to topics.
- Topics are an ordered list of events. The topics are stored in non-volatile storage and may be replicated.

## Consumer API
- The consumer of Kafka subscribes to a topic.

## Streams API
- A stream can consume data from producers and transform the data in real-time and produce the transformed data to consumers.
