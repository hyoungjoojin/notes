# OSI Model

### Layer 4: Transport Layer
- Provides data transfer between end hosts.
- Possible services the transport layer can provide to applications include:
  - Reliable data transfer
  - Error recovery
  - Data sequencing
  - Flow control
- Provides Layer 4 addressing via port numbers.
  - Port numbers can be used for session multiplexing and identifying Layer 7 protocols.
- TCP and UDP are examples of [Layer 4 protocols](./l4.md).
- The Layer 4 PDU is called a segment.
