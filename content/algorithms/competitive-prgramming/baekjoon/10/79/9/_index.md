# 쇠막대기

> Level: Silver 2

## Problem

Given a string composed of parentheses, the following rules apply.

- Adjacent matching parentheses represent a laser pointing down.
- All other matching parentheses represent a block.

If all lasers can reach the blocks and cut them, count the number of block parts generated.
