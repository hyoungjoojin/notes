# 카드 구매하기

> Level: Silver 1 \

## Problem

Given an array $P$ and an integer $n$, find the maximum of $\sum_{j}{P_{s_j}}$ where $0\leq s_j \leq n$
and $\sum_{j}{s_j} = n$.

## Solution

### Dynamic Programming
