# 절댓값 힙

> Level: Silver 1

## Problem

For an array, we can either insert the number or pop out the number with the smallest absolute value
in the array. Given a list of commands, apply the commands to an empty array.

## Solution

### `std::priority_queue`

> Implementation: [C++](./src/a.cpp) (12ms, 2532KB) \
> Time: $O(n\log n)$, Space: $O(n)$
