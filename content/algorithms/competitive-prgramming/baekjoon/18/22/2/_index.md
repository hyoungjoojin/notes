# 투에-모스 문자열

> Level: Silver 2

## Problem

A sequence $s$ is made by continously applying the following rules.

1. The initial sequence is $0$.
2. For each iteration, we flip the bits of $s$ and append that value to $s$.

Given an index $i$, find $s_i$.
