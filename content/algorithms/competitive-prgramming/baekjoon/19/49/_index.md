# 우수 마을

> Level: Gold 2

## Problem

Given a tree $T$ where each node of $T$ has a value $v[i]$, find the maximum sum of values such that no two adjacent nodes
are selected and every node is either selected or has at least one adjacent selected node.

## Solution

### DFS + Dynamic Programming
