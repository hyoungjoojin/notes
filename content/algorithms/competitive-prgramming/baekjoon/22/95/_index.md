# 세 수의 합

> Level: Gold 4

## Problem

Given a set $U$, find 4 elements $a, b, c, k$ (the elements may be the same) within $U$ such that $a + b + c = k$.

## Solution

### Binary Search

> Implementation: [C++](./src/a.cpp) (60ms, 14448KB) \
> Time: $O(n^2\log n)$, Space: $O(n^2)$
