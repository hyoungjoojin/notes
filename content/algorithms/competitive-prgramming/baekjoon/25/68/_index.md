# 전깃줄 - 2

> Level: Platinum 5

## Problem

Given an array of integer pairs, each pair represents a line between point $(0, p_1)$ and $(1, p_2)$.
Return the list of lines to remove so that the remaining lines do not cross each other and the number of lines removed is
minimized.
