# Missing Number

## Problem

Given an integer $n$ and an array $A$ of size $n - 1$ and $\forall a_i\in A ,a_i \in [1, n]$, find the
number within the range $[1, n]$ that is not in $A$.

## Solution

### Counting Sort

> Implementation: [C++](./src/a.cpp) \
> Time: $O(n)$, Space: $O(1)$
