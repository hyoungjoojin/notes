# Binary Exponentiation

Binary exponentiation is an algorithm for computing powers using $O(\log{n})$ multiplications instead
of $O(n)$ multiplications.

Binary exponentiation can work for any repeated operation that satisfies associativity.

## Applications

- Compute $a^b \mod m$.
