# Email Protocols

Email protocols can be categorized into two major types: protocols for sending
messages and protocols for accessing messages. SMTP is the protocol for sending
messages, while IMAP and POP3 are the protocols for accessing messages.

## SMTP

SMTP (Simple Mail Transfer Protocol) is a protocol for sending email messages
from one client to another.

SMTP restricts the data of email messages to 7-bit ASCII characters. Therefore,
all email messages must be encoded into ASCII before being sent.

## IMAP

IMAP (Internet Message Access Protocol) is a protocol for accessing and managing
email messages on multiple devices. IMAP is useful when emails have to be
synchronized across multiple devices using a single central mail server.

## POP3

POP3 (Post Office Protocol version 3) is a protocol for retrieving email
messages from a mail server and storing them on a local device. POP3 is useful
when emails have to be accessed when the user has limited to no internet
connectivity, and the user does not need to synchronize emails across multiple
devices.
