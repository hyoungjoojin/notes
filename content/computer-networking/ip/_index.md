# IP (Internet Protocol)

The Internet Protocol (IP) is a network layer protocol that provides a
best-effort delivery service.

## IP Versions

IPv4 and IPv6 are the two versions of IP, and the numbers indicate the version
of the protocol. Other versions did exist, but 1, 2, and 3 were drafts and 5 was
an experimental version.
