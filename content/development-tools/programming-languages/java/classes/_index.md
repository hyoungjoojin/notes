# Java Classes

## Visibility

Variables and methods of a class in Java can have their visibility mutated using access modifiers.
Classes within the same package of a class are seen as friendly and can see package-visible
variables and methods.

| Modifier              | Visibility                        |
| --------------------- | --------------------------------- |
| private               | Only self                         |
| No Modifier (default) | Same package classes              |
| protected             | Same package classes + Subclasses |
| public                | All classes                       |
