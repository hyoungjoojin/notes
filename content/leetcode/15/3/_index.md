# Find Minimum in Rotated Sorted Array

> Level: Medium \

## Problem

Given an array $A$ where $A$'s elements are all unique and $A$ is a rotated version of a
sorted array, find the minimum of $A$.

## Solution

### Binary Search

> Time: $O(\log n)$, Space: $O(1)$
