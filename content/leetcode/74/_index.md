# Search a 2D Matrix

> Level: Medium \
> Tags: Binary Search

## Problem

Given a 2D array of numbers where the numbers are sorted such that each row is sorted and
the first integer or one row is greater than the last integer of the previous row,
find whether or not a given target variable is in the array.
