## Network Programming in C

Sockets in Unix are a way for programs to communicate with each other. Internet
sockets can be used for network programming.

There are a few different types of Internet sockets. Stream sockets
(`SOCK_STREAM`) are used for TCP connections, while datagram sockets
(`SOCK_DGRAM`) are used for UDP connections.
