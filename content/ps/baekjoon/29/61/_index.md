# 도영이가 만든 맛있는 음식

> Level: Silver 2

## Problem

Given an array of $(s, b)$, find the minimum value of $|S - B|$, where $S$ is the product of all $s$ values and $B$ is the
sum of all $b$ values for a non-empty subset of the array.
