# JOSE (JSON Object Signing and Encryption)

The JOSE framework is a set of specifications for providing a standard way of signing and
encrypting digital content using JSON formats.

The JOSE framework includes the following specifications.

- [JWT (JSON Web Token)](./jwt)
- [JWS (JSON Web Signature)](./jws)
- [JWE (JSON Web Encryption)](./jwe)
- [JWK (JSON Web Key)](./jwk)
- [JWA (JSON Web Algorithms)](./jwa)
