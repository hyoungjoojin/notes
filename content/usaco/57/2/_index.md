# Breed Counting

> Level: Silver \

## Problem

Given an array $A$ where each element of $A$ is within the set $\lbrace 1, 2, 3 \rbrace$,
and $N$ queries of $(s, e)$, find the number of occurrences for 1, 2, and 3 in $A_{s:e}$.

## Solution

### Prefix Sum

> Implementation: [C++](./src/a.cpp) \
> Time: $O(n)$, Space: $O(n)$
