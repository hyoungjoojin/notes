# Chapter 1: Concurrency and Asynchronous Programming: A Detailed Overview

## History of Multitasking

- Non-preemptive Multitasking: Application developers need to give up the processor usage.
- Preemptive Multitasking: The operating system takes the processor from a running program.
- Hyper-Threading: A physical CPU core is split into multiple logical cores.
- Multi-core Processors: Multiple CPU cores are used to perform multitasking.

## Concurrency and Parallelism

- Concurrency is about dealing with a lot of things at the same time.
- Parallelism is about doing a lot of things at the same time.
